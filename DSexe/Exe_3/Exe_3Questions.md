##  题目 1：有效括号字符串判断   

**任务描述**：
 给定一个只包含字符 `'('`, `')'`, `'['`, `']'`, `'{'`, `'}'` 的字符串 $S$，判断该字符串是否有效   。

**判定标准**：
*  **类型匹配**：左括号必须用相同类型的右括号闭合   。
*  **顺序正确**：左括号必须以正确的顺序闭合   。

##  题目 2：双栈实现队列   

**任务描述**：
 请仅使用**两个栈**实现一个先入先出（FIFO）队列   。


**支持操作**：
 该队列应当支持标准队列的所有核心操作   ：

| 操作 | 描述 |
| :--- | :--- |
| `push(x)` |  将元素 $x$ 推到队列末尾   |
| `pop()` |  从队列开头移除并返回元素    |
| `peek()` |  返回队列开头的元素   |
| `empty()` |  如果队列为空，返回 `true`；否则返回 `false`   |

##  附加题：下一个更大元素   

**任务描述**：
 给定两个没有重复元素的数组 `nums1` 和 `nums2`，其中 `nums1` 是 `nums2` 的子集   。

**求解要求**：
*  找出 `nums1` 中每个元素在 `nums2` 中的**下一个更大元素**   。
*  **定义**：数字 $x$ 的“下一个更大元素”是指 $x$ 在 `nums2` 中对应位置右边第一个比 $x$ 大的元素   。
*  **特殊情况**：如果不存在更大元素，则该位置输出 `-1`   。

**算法效率**：
*  **时间复杂度**：要求达到 $O(n+m)$   。
